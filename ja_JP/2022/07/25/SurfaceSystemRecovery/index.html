Microsoft Surfaceリカバリの究極のチュートリアル

The article is translated by machine, for better reading experience, see the original one: The ultimate tutorial of Microsoft Surface recovery


記事は機械で翻訳されています。読みやすくするために、元の記事を参照してください：The ultimate tutorial of Microsoft Surface recovery



多くの Microsoft Surface ユーザーによると、オペレーティング システムは簡単にクラッシュします。では、問題をどのように処理しますか？チュートリアルでは、3 つの方法を紹介します。

警告:

1. このチュートリアルに含まれる次の内容は、参照のみです。 コア テクノロジの内容については、Microsoft の公式ドキュメントを参照してください。

2. チュートリアルを読んでいる間は、一歩一歩進んではいけません。ステップバイステップで読む必要があります。

3. 以下の手順を実行する前に、システムの回復中は USB フラッシュ ディスクを取り外さないでください、電源を切る、電源ボタンを長押しして強制シャットダウンを行います。
さらに、すべての重要なファイルを安全な場所にバックアップしたことを確認してください。

4. デバイスが現在正常に動作しているかどうかに関係なく、Bitlocker キーを忘れた場合は、このチュートリアルに従わないでください。

5. Microsoft の公式ドキュメントについては、次の URL にアクセスしてください：Surface for Windows を復元またはリセットする。 また、このチュートリアルの次の内容は読まないでください。


方法1

組み込みのWindows回復機能でデバイスを回復する


警告:

お使いのデバイスが Windows 10 バージョン 1803 を実行していないことを確認してください。

現在の Windows バージョンを確認するには、Windows を押してくださいロゴ キーと R を同時に入力し、テキスト フィールドに Winver と入力して OK ボタンをクリックするか、 を押します。 Enter。
マークされた領域はシステム バージョンを示します。

バージョン 1803 ユーザーの場合は、方法 2 または以下の 方法 3。



Windows 10 ユーザー

  • 設定アプリを開き、「更新とセキュリティ」ラベルをクリックします。


  • [リカバリラベル]をクリックし、[このPCラベルをリセット]の下の[開始]をクリックします。


  • 次の内容を Windows 11 ユーザーとして読んで、手順を続行します。

Windows 11 ユーザー

  • 設定アプリを開き、[リカバリ]を見つけてクリックします。


  • [この PC をリセット] ラベルのすぐ内側にある [PC をリセット] ボタンをクリックします。


  • 回復する方法を選択してください。


    • 個人用ファイルはどれですか？
    個人用ファイルには、デスクトップドキュメントダウンロード音楽などのユーザーフォルダ内のファイルが含まれます写真ビデオ

  • 再インストール方法を選択します。通常、ローカル再インストールをお勧めします。その方が高速です。


    • 警告：
    Windowsインストールメディアを要求された場合は、クラウドダウンロードオプションを選択するか、方法3でSurfaceリカバリイメージを使用できます。.

  • デフォルト設定を使用する場合は、[次へ] をクリックしてください。


  • 変更する場合は、[設定の変更] ラベルをクリックし、変更を加えて [確認] ボタンをクリックします。


  • デバイス上のすべての重要なファイルが安全な場所にバックアップされていることを確認し、準備が整ったら [リセット] ボタンをクリックします。


  • デバイスは数回再起動し、最終的にWindows11の新しいコピーを取得します。

方法2

WindowsISOファイルを使用してオペレーティングシステムをすばやく修復します

Windows 10の一部のバージョンは、組み込みのWindows回復機能を使用した後に壊れます。

デバイスがインターネットに接続されていることを確認してください。
この方法は、Windows 10 と Windows 11 の両方で機能します。

  • タスクバーのアイコンをクリックして、Windowsエクスプローラーを開きます。


    • クリックしても開けない場合は、Ctrl Alt と Del を同時に押して、タスク マネージャー をクリックします。 .
    [タスク マネージャー] ウィンドウで、[ファイル] をクリックし、[新しいタスクの作成] をクリックします。
    テキスト フィールドに エクスプローラー と入力し、 OK ボタンをクリックするか、Enter を押します。


  • Local Disk（C :) と入力し、次に Program Filesフォルダと入力します。


  • Internet Explorer フォルダに入ります。


  • ieexplorer.exe を見つけて実行します


  • リンクを開く Windows 10 のディスク イメージ (ISO ファイル) のダウンロード or Windows 11 をダウンロードする InternetExplorerを使用。



    • ダウンロードする64ビットイメージを選択します。

  • ダブルクリックしてisoファイルをマウントし、 Setup.exeを実行します。


    • すべてを保持を選択します。

    デバイスが数回再起動します。

    方法3

    SurfaceRecoveryImageを使用してデバイスを回復します

    要件:

    1. インターネット接続と、Windows 10 以降を実行している PC があることを確認します。

    2. 容量が 16GB 以上の USB 3.0 フラッシュ ディスクがあることを確認します。 (これはダウンロードするイメージ次第です。リカバリ イメージの中には、8GB 未満のスペースしか必要としないものもあれば、16GB 以上を必要とするものもあります。)

    • ブラウザでリンクを開きます：Surface 回復イメージのダウンロード

    • ページの指示に従って画像をダウンロードします。


    • ダウンロード中に、USBフラッシュドライブを挿入し、検索バーに「リカバリドライブの作成」と入力します。


    • [システムファイルをリカバリドライブにバックアップする]の選択を解除し、[次へ]をクリックします。
      リカバリドライブとして使用するドライブを選択し、[]をクリックします。次へ。
      しばらく待って、プロセスが終了するのを待ちます。
    • ダウンロード プロセスが完了したら、クリックして開き、その中のすべてのファイルを作成した回復ドライブにコピーします。

    重複ファイルが存在するという警告が表示された場合は、置換オプションを選択してください。

    • Surfaceデバイスが完全にシャットダウンしたら、リカバリドライブを挿入し、SurfaceUEFIに読み込まれるまで電源ボタンと音量大ボタンを同時に押し続けます。


      • 警告:
      UEFI パスワードを設定していた場合は、それを使用して Surface UEFI に完全にアクセスしてください。
      パスワードを忘れてデバイスが Windows を起動できない場合は、Microsoft サポート担当者に連絡してください。
      />それを忘れたが、幸運にも最初の起動順序として USB ドライブを設定している場合は、この手順をスキップできます。

    • Surface UEFI で、[ブート構成] を選択します
      • 左側のラベルをクリックし、Windows ブート マネージャーに USB ストレージ をドラッグします。


      上の図に示すように、ブート構成ロックを有効にするが無効になっていることを確認してください。

    • 左側の[終了]ラベルをクリックしてから、[今すぐ再起動]ボタンをクリックします。


    • デバイスは自動的にリカバリドライブから起動します。
      言語とキーボードレイアウトを選択します。


      使用可能な言語は、Surfaceデバイスを購入した地域によって異なります。


    • トラブルシューティングオプションを選択します。


    • 次に、[ドライブから回復]を選択します。

      オプションが表示されない場合は、考えられる理由と解決策を次に示します。

      1.リカバリドライブを作成したときに問題が発生しました。上記の手順に従って再作成してください。

      2.一部のUSBドライブは、USB 3.0のものであっても、Surfaceデバイスで認識できないため、リカバリドライブとして使用できません。他のものと交換する必要があります。

      Kingston DataTraveler100G3を選択することをお勧めします。

      3.一部のオンラインブログでは、|リカバリドライブを作成する|の手順をスキップする場合があります。リカバリドライブをFAT32としてフォーマットしてから、リカバリファイルをそのままコピーします。それは時々動作しますが、多くの人々がドライブからの起動に閉じ込められていました。したがって、この方法をとるべきではありません。


    • デバイスで Bitlocker 機能を有効にしている場合は、同様のページが表示されます。また、復元を完了するにはロック解除キーが必要です。

      通常、Microsoftアカウントに保存されています。他のデバイスの画面でリンクを開くだけで、元に戻すことができます。


    • 利用可能なオプションは2つあります。ファイルを削除するだけとドライブを完全にクリーンアップするです。 ファイルを削除するオプションを使用すると、回復速度が速くなりますが、ドライブが完全にクリーンアップされるわけではありません。以前の個人ファイルが他の人によって回復されないように、回復後にデバイスを自分で保持しない場合は、ドライブを完全にクリーンアップするを選択します。


    • リカバリを開始する前に、処理中に終了できないため、デバイス上のすべての重要なファイルをバックアップしたことを再度確認してください。準備ができたら、[回復]ボタンをクリックします。
