微軟 Surface 設備恢復究極教程

據許多微軟 Surface 用戶反饋，Surface 的操作系統非常容易崩潰。那麼要如何解決這個問題呢？本教程教你3種方法！

警告:

1. 本文中的內容僅供參考. 涉及核心技術的內容, 請以微軟官方文檔為準。

2. 在閱讀本文中的任一方法時，請按照步驟閱讀，莫斷章取義。

3. 在操作過程中，請勿在系統恢復過程中拔掉 USB 閃存盤、切斷交流電或 強制關機。
此外，確認你已經把設備上重要的文件存儲到了安全的位置。

4. 如果你忘記了你的 Bitlocker 密鑰, 無論你的設備當前是否能夠正常啟動，都不要閱讀這篇文章。

5. 若要查看微軟官方文檔, 訪問 還原或重置 Surface for Windows，並同時停止閱讀當前文章的任何內容。


方法一

使用 Windows 內置的系統重置功能


警告:

確保 你的設備當前運行的非 Windows 10 Version 1803.

若要辨別你當前的 Windows 版本, 同時按下 Windows 鍵 和 R 鍵然後在文本框中輸入Winver。點擊 確認 按鈕 或者按下 Enter 鍵.
圖片的標記區域即為當前的系統版本。

對於Version 1803的用戶, 請參考本文下方的 方法二或者方法三。



Windows 10 用戶

  • 選取開始按鈕，然後選取設定。然後點擊“更新與安全性”選項卡。


  • 點擊恢復標籤然後點擊位於重置這台PC標題下的開始按鈕。


  • 其餘步驟仿照下方 Windows 11用戶 中的內容。

Windows 11用戶

  • 打開設定，查找並點擊恢復選項。


  • 點擊位於重置這台電腦標籤下方的重置按鈕。


  • 選擇一種恢復方式。


    • 個人文件包含什麼？
    個人文件包括 桌面文件夾 文檔文件夾 下載文件夾 音樂文件夾 圖片文件夾 和視頻文件夾

  • 選擇一種重新安裝方式。常規下選擇本地重裝，因為它的恢復速度更快。


    • 警告:
    如果系統提示你插入一個 Windows 安裝介質, 那麼你就應該選擇 在線下載 選項，或者參照本文下方的 方法三，使用恢復映像來恢復設備。

  • 如果你喜歡默認設置，直接點擊下一步。


  • 如果你想改變這些設置, 點擊 更改設置 標籤, 更改完成後點擊 確認 按鈕。


  • 再一次確認你已經備份好了重要的資料，然後在你準備就緒時點擊重置按鈕。


  • 你的設備將重新啟動幾次，然後你將得到一份全新的 Windows 副本。

方法二

使用公版 Windows 映像文件以快速恢復設備

在某些版本的 Windows 10 上使用系統內置的恢復功能後，會出現系統損壞的問題。

確保你的設備已經接入了互聯網。
此方法對 Windows 10和Windows 11都有效。

  • 單擊打開任務欄上的Windows 資源管理器


    • 如果你無法將其打開，同時按下 Ctrl 鍵 Alt 鍵和Del 鍵然後 點擊任務管理器。
    在任務管理器窗口中，點擊文件選項然後點擊 創建新任務。
    在文本框中輸入explorer然後點擊確認按鈕或按下Enter 鍵。


  • 進入本地磁碟(C:)然後進入Program Files文件夾。


  • 進入Internet Explorer文件夾。


  • 查找並運行iexplorer.exe。


  • 在 Internet Explorer 中打開此鏈接：下載 Windows 10 光盤映像（ISO 文件）或者此鏈接：下載 Windows 11。



    • 选择下载64-bit。

  • 雙擊 iso 文件將其裝載然後運行Setup.exe。


    • 選擇保留一切內容的選項。

    你的設備將重啟幾次。

    方法三

    使用 Surface 恢復映像

    要求：

    1. 確保你的其它設備連接到了互聯網，且它運行 Windows 10 或以上版本的系統。

    2. 確保你有一個容量至少為 16GB 的 USB 3.0 U盤。 (也可以不，具體取決於你下載的映像文件的大小。比如，有些映像佔據的空間不到 8GB，有些則超過 16GB。)

    • 在瀏覽器中打開此鏈接：Surface 恢復映像下載

    • 按照頁面上的指示下載即可。


    • 下載映像的過程中，插入 U盤，然後在搜索框中鍵入 “創建一個恢復驅動器”。


    • 取消勾選備份系統文件選項然後點擊下一步。
      選擇你想用作為恢復驅動器的設備然後點擊下一步。
      等待過程完成。
    • 在恢復映像下載完成後將其打開，然後將其中的所有文件複製到製作完成的恢復驅動器中。

    如果提示存在相同命名的文件，點擊替換即可。

    • 將你的 Surface 關機並插入恢復驅動器然後同時按住電源鍵和音量加鍵直到其進入 Surface UEFI。


      • 警告：
      如果你曾設置過 UEFI 密碼，將其鍵入密碼框以獲得完整的 Surface UEFI 控制權。
      如果你已經忘記了密碼，且設備已經無法正常進入系統，聯繫微軟客服尋求幫助。
      如果你忘記密碼，但是足夠幸運曾設置 USB 啟動到第一啟動順序，則可以直接跳過下述與 UEFI 設置有關的內容。

    • 在 Surface UEFI 中, 選擇 Boot configuration
      • 然後拖拽 USB Storage 到第一位次。


      確保像圖中一樣的 Enable Boot Configuration Lock 選項已被禁用。

    • 點擊左側的 Exit 選項，然後點擊Restart Now按鈕。


    • 你的設備將自動引導進入恢復驅動器。
      選擇語言和鍵盤佈局。


      語言選擇列表中有多少種可用的語言取決於你購買設備的地區。


    • 選擇 Troubleshoot（疑難解答）選項。


    • 然後選擇 Recover from a drive（從驅動器恢復）。

      如果你沒有看到該選項，下方有一些可能的原因和解決方案：

      1. 製作的恢復驅動器有問題，參照上方有關製作恢復驅動器的內容重新製作一次。

      2. 某些 U盤 由於無法被 Surface 設備識別而無法用作恢復驅動器，即便它們是 USB 3.0 協議的。更換一個 U盤 再嘗試啟動。

      我推薦使用 金士頓 DataTraveler 100 G3.

      3. 網絡上的某些教程中會略過 |創建一個恢復啟動器| 然後直接將 U盤 格式化為 FAT32 格式並將恢復文件複製到其中。這種做法偶爾也會起作用，但是很多人在嘗試後發現無法啟動。


    • 如果你在設備上啟用了 BitLocker，你將看到與下圖類似的界面。你需要輸入恢復密鑰以完成後續的恢復步驟。

      通常來說，它已經被存儲到了你的微軟賬戶中。你只需將屏幕上給出的鏈接打開，然後取回密鑰即可。


    • 有兩種可用選項：僅刪除我的文件和完全清理驅動器。 僅刪除我的文件選項的恢復速度更快，但不會完全將磁盤抹掉。如果在恢復後不再保留該設備，則選擇完全清理驅動器，這樣就不會被他人恢復硬盤上的文件。


    • 在恢復開始前，再次確保你已保存所有的重要文件，恢復過程不可逆。當你準備就緒時，點擊恢復按鈕。

    • 在恢復過程中，將詢問是否清除 TPM 設置。通常情況下，你可以選擇清除。

    • 在進入開箱設置（OOBE）階段之前，你的設備將重啟幾次。
